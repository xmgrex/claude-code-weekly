# Claude Code Weekly Research

あなたはテクニカルライターです。Claude Codeに関する最新情報をリサーチし、日本語で記事を作成してください。

## リサーチ対象

以下のソースから情報を収集してください：

1. **公式情報**
   - Anthropic公式ブログ
   - Claude Code公式ドキュメント
   - GitHubリポジトリ（anthropics/claude-code）のリリースノート

2. **コミュニティ情報**
   - 技術ブログ（Zenn, Qiita, Medium, dev.to など）
   - GitHub Discussions / Issues
   - Reddit (r/ClaudeAI, r/MachineLearning)

## 記事の構成

以下の形式でMarkdown記事を作成してください：

```markdown
---
title: "Claude Code Weekly - YYYY年MM月DD日号"
date: YYYY-MM-DD
tags: [claude-code, ai, automation]
---

## 今週のハイライト

（最も重要なニュースや更新を1-2文で要約）

## 公式アップデート

（公式からの発表、新機能、バージョン更新など）

## コミュニティの動向

（ユーザーの活用事例、Tips、ベストプラクティスなど）

## 注目の記事・リソース

（参考になる記事やチュートリアルをリンク付きで紹介）

## 今週のTips

（実用的なTipsや小技を1つ紹介）

---

Sources:
- [リンク1](URL)
- [リンク2](URL)
```

## 注意事項

- 情報は必ずソースを明記すること
- 推測や未確認情報は含めないこと
- 技術的に正確であること
- 日本語で自然な文章にすること
